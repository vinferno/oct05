CREATE: takes a color and creates a list item.
READ: takes an array and creates multiple items.
UPDATE: takes the active li (green border) and updates color. OR: can take a comma seperated array with two values of colors and change all li's with first value to second value OR: can take a comma seperated array with two values of a number and a color and change the numbered li to second color
DELETE: will delete the active li OR: a specified color OR: number.
CHOOSE FILE: will take any chosen text file(txt,csv,json,ect.) with a comma seperated array and load it into the "Read" input.
UP/DOWN arrow to change acive li.
DELETE button: will delete active li
List Itemes on hover: CLICK on '-' button will delete li CLICK on 'Rename will focus on update.
URL input: Enter a url of an array and it will load in the Read input box
